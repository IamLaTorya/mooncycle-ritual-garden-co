🌙 Mooncycle Ritual Garden Co.
A multi-page, lunar-inspired website blending botanical aesthetics with modern web standards.

🌿 Overview
Mooncycle Ritual Garden Co. is a responsive e-commerce concept focused on ritual tools and spiritual gardening. It features a cohesive brand design, semantic structure, and interactive product displays.

🔧 Tech Stack
Structure: HTML5
Styling: CSS3, Bootstrap 5
Icons: Bootstrap Icons

✨ Features
Responsive Design: Mobile-first layout using Bootstrap utilities.
Product Catalog: Custom cards with CSS-only Quick View modals.
Accessible: Built with semantic headings and descriptive alt-text.
Animations: Smooth scroll effects and a botanical-spiritual UI.

📁 Pages
Home: Brand introduction and featured collections.
About: Mission and lunar philosophy.
Catalog: Full product list with interactive modals.
Blog: Articles on cycles and gardening.
Contact: Inquiry form and community links.
Privacy: Standard data protection.
